\chapter{社团}
\section{社团活动记录}
社团活动记录，是开发建平中学学生平台的最初目的。

因为在每次社团活动过后，学校的每个大大小小的社团都需要上交社团活动记录。但随着社团数量的不断上升，人工使用电子邮箱或者QQ收取社团活动记录的时间成本及体力成本也日益攀高。
因此，建平中学社团部与建平中学JEEK信息社进行了合作来开发建平学生平台，意在使用互联网信息技术，建立在线社团活动记录提交平台，减少不必要浪费的人力成本。

社团活动记录的内容如下
\begin{itemize}
  \item 社团名称及社团序号
  \item 社团联系人姓名、班级、年级
  \item 社团联系人联系方式（电话、QQ）
  \item 社团活动开始时间及结束时间
  \item 社团活动地点
  \item 社团活动过程
  \item 社团互动内容
  \item 社团活动评价
  \item 社团活动感悟
\end{itemize}
\section{社团活动记录统计}
社团部会于每一学期的期末根据社团一学期的活动进行打分。打分会影响金苹果社团的评比。
